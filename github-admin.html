32Gamers GitHub Admin

Setup Instructions:

  1. Create a GitHub repository for your 32Gamers portal
  2. Upload your site files to the repository
  3. Go to GitHub Settings → Developer settings → Personal access tokens
  4. Create a token with 'repo' permissions
  5. Enter your details below to manage your apps

Benefit: Your apps will be stored in GitHub and automatically update your live site!
